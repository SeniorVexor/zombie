## 🇵🇭 رول دفاع فیلیپین صفحه اول 🇵🇭

**آمادگی و روحیه:**  
ما به ۱۶M سرباز روحیه کامل می‌دیم: سخنرانی رهبر → قول پیروزی ۱۰۰%، خانواده‌ها امن در پناهگاه‌ها. تجهیزات (پهپاد/مین/موشک/گنبد) آماده. شعار: "دشمن هرگز ساحل نبینه – ما اول می‌زنیم!"

**روحیه مردم/امنیت:**  
قول پیروزی به مردم، پناهگاه‌های هولوگرامی ضد اتمی/هک در Manila/Davao/Cebu (غیرقابل نفوذ). سربازان نزدیک مردم → سرکوب شورش/جاسوس فوری. رهبر حس شکست صفر می‌کنه.

**جلوگیری هک/سایبری:**  
۱۰۰ هکر ۲۴/۷ IP ناشناس سرکوب. سیستم هولوگرامی/ماهواره ضد هک (فقط مقامات). TV/گوشی تحت نظر → مردم آگاه، راز لو نمی‌ره. بانک/دفاع سایبری امن‌ترین.

**دفاع کلی: "دیوارهای اقیانوسی آهنین باتان" (جامع vs همه حملات)**  
**دیوار آهنین ساحلی (ضخامت ۱۵m، طول ۶۰m هر بخش):** دور تمام سواحل/خلیج‌ها (Lingayen/Lamon/Surigao + جزیره‌ها) با آهن معادن. بالای دیوار: اسنایپر/توپخانه/نیرو ویژه. زیر: مین سنسوردار (انسان/تانک/زیردریایی).  
**Position دقیق:**  
- **Lingayen Gulf (شمال Luzon - حمله دریایی اصلی):** دیوار ۵۰km، ۵M سرباز، ۵۰ مین دریایی، ۳۰ گنبد.  
- **Lamon Bay (شرق):** دیوار ۳۰km، ۳M سرباز، ۳۰ مین دریایی، ۲۰ گنبد.  
- **Surigao Strait (جنوب Mindanao):** دیوار ۴۰km، ۳M سرباز، ۲۲ مین دریایی، ۲۰ گنبد.  
- **سایر سواحل/جزیره‌ها (Visayas/Mindanao):** ۵M reserves + guerrilla hubs.  

**زمینی:** ۱۶M → گشت ۲۴/۷، پایگاه هر ۵km choke point.  
**هوایی/پهپادی:** ۱۲k پهپاد → swarm bombers/UAV.

**پدافند:** ۱۰۰+ گنبد (S400-like، ۱۲ لانچر) → ۹۹% intercept موشک/بمب/هوابرد.  
**vs زیردریایی:** مین عمقی + پهپاد anti-sub.

## 🇵🇭 رول دفاع فیلیپین صفحه دوم 🇵🇭

**شروع حمله:** پهپاد جاسوسی → مکان ناو/هواپیما/پایگاه دشمن → موشک/پهپاد پیش‌دستانه برای خنثی‌سازی.  

**ماهواره/دوربین هولوگرامی:** ۱۰۰% پوشش → جاسوس/دشمن فوری به سربازان.  
**محافظ هولوگرامی:** ضد ماهواره دشمن → نیروها مخفی. گنبد موشک‌ها نابود.  

**مین‌ها:** ۱۰۰۵+ زمینی (دور سواحل/جنگل/زیر دیوار) + ۱۰۲+ دریایی (مارپیچ: الگوی labyrinth → ناوها گیر/غرق).  

**لایه‌های دفاعی (جبهه‌بندی ۳ لایه + مخفی - vs زمینی/هوابرد):**  
هر لایه ۶ گروه A-F (فاصله ۴km مین). لایه‌ها ۸km فاصله.  
- **لایه جلو (ضعیف - تضعیف):** ۴M سرباز + مین/خندق → ۳۰% تلفات اولیه. Position: ۱km داخل ساحل.  
- **لایه وسط (قوی):** ۶M + ۴k پهپاد → encirclement/backstab. Position: ۹km داخل.  
- **لایه عقب (قدرتمند):** ۴M reserves + توپخانه → grinder. Position: ۱۷km داخل (نزدیک شهرها).  
- **لایه مخفی (جلو ۸km + ۴km مین):** ۲M زیرزمینی jungle → غافلگیری پشت (۵۰% تلفات).  
اگر چند جهت حمله → گروه‌ها کمک. پایگاه‌ها: سرباز/مین کافی.  

**خندق‌ها:** جلوی لایه‌ها (پوشیده برگ/خاک نرم) → سیم خاردار برقی + مین → زمان‌کشی.  

**فریب:** استتار → fake retreat به jungle → تله encirclement. تونل باتان ۲.۰ (Luzon mountains): flamethrowers/traps زیرزمینی.  

**دفاع شهرها (جدول Position):**

## 🇵🇭 رول دفاع فیلیپین صفحه سوم 🇵🇭

**تله jungle/mountains (خلاقیت vs زمینی):**  
- **بوفالو بمب‌دار:** karabao محلی + C4 → رم به دشمن → انفجار (Mindanao forests).  
- **خندق مرگ:** پوشیده → افت → برق/مین (Luzon trails).  
- **کمین guerrilla:** ۴M ranger → hit-run/sabotage supply. Position: هر مسیر jungle ۲km.  

**اصول پهپادی/هوایی (مانورها vs هوایی/هوابرد):**  
گروه: ۵k پهپاد → هوا-هوا انتحاری.  
**مانور Hook & Break:** موشک دشمن → دور موتور بالا → تیز راست/چپ → low altitude swarm below.  
**مانور فایت نزدیک:** پیچ منحرف‌کننده → مخالف پیچ → تیررس پشت → مهاجم شو.  
**تاکتیک:** ۲ پهپاد bait جلو → بقیه پشت → نابود.  

**خنثی‌سازی حمله‌های دشمن (Counter-Attack پیش‌دستانه - خلاقیت ۱۰۰%):**  
- **vs هوایی/پدافند:** گنبد آهنین + ۵۰ موشک پدافندی → ۹۹% intercept bombers/UAV قبل رسیدن. پهپاد جاسوسی مکان فرودگاه/پایگاه دشمن پیدا → ۳k پهپاد انتحاری swarm به runways → نابود باند/جنگنده‌ها روی زمین (نتونن بلند شن). موشک ساده (۵۰۰) point به hangars → انفجار سوخت/مهمات.  
- **vs دریایی/زیردریایی:** پهپاد anti-sub + مین عمقی → غرق subs قبل launch. ۴k پهپاد به ناوگان/پایگاه دریایی دشمن → kamikaze carriers → ۸۰% fleet disabled. موشک anti-ship (۵۰۰) به docks → جلوگیری از sortie.  
- **vs زمینی/هوابرد:** guerrilla spies → مکان camps/landing zones → موشک (۴۷) + پهپاد swarm پیش‌دستانه به transports/troops → ۵۰% تلفات قبل deploy. توپخانه ساحلی (A-222-like) → شلیک لیزری به parachutes/landing craft.  
- **vs اتمی/موشکی:** جاسوسان → مکان silos/subs → پهپاد/موشک preemptive strike. گنبد + هولوگرامی decoys → intercept ۱۰۰%.  
**Position خنثی‌سازی:** از Manila (HQ) launch → پهپاد swarm ۱۰۰km range به پایگاه‌های نزدیک (e.g., Taiwan/China-like در بازی).  

**رادار:** دور جزیره‌ها + زیر دیوار/گنبد → ۱۰۰% تشخیص UAV/جاسوس.  

**جاسوسان:** پلیس مخفی + حرارتی → سرکوب. پهپاد جاسوسی مرزها → جهت حمله فوری.  

**نقش موشک/بمب:** ۱۰۴۷ موشک → point به transports/تانک/پایگاه‌ها. پس هوایی → سنگرشکن beaches. لانچر ۵km داخل.  

**ساختمان‌های آتلانتیکی ساحلی (beaches):** توپخانه خودکششی + لیزر.
